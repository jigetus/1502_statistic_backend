REST API Школы №1502 "Энергия", корпус Гамма

Данный сервис предоставляет возможность получения статистических сведений по успеваемости учеников школы №1502, корпус Гамма. Сервис не предоставляет возможности получения личных данных отдельных учеников, а так же их оценок.

Получение списка учебных групп по названию класса

GET /getunitsubjects
Обязательный QUERY параметр: unit

Пример запроса:

/getunitsubjects?unit=11-Л

Ответ сервера:

Массив объектов, имеющих следующую структуру:
{
"unit":"11-Л",
"group_id":5797383,
"group_name":"Технология использования современных измерительных приборов и оборудования 11Л1  (Гамма)",
"subject_id":18930,
"subject_name":"Технология использования современных измерительных приборов и оборудования"
}

Получение средней оценки по главным предметам по названию класса

GET /mediumsubjects
Обязательный QUERY параметр: unit

Пример запроса:

/mediumsubjects?unit=11-Л

Ответ сервера:

Массив объектов, имеющих следующую структуру:
{
"avg_mark":3.4558823529411766,
"group_id":[5797398],
"title":"Алгебра",
"id":[46,117],
"unit":"11-Л"
}

Получения лучшего класса в предмете по среднему баллу

GET /bestunitsinsubjects
Обязательный QUERY параметр: korpus

Пример запроса: (на данный момент доступен только корпус Гамма)

/bestunitsinsubjects?korpus=gamma

Ответ сервера:

Массив объектов, имеющих следующую структуру:
{
"max_mark":4.27059,
"title":"Русский язык",
"unit":"7-Л"
}

Получение ТОП-10 классов в корпусе по среднему баллу всех предметов

GET /top10avg
Обязательный QUERY параметр: korpus

Пример запроса: (на данный момент доступен только корпус Гамма)

/top10avg?korpus=gamma

Ответ сервера:

Массив объектов, имеющих следующую структуру и отсортированных в порядке убывания 
{
"AVG":4.39295848778316,
"unit":"8-Н"
}
